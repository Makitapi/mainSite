# Tech Context

## Technologies Used
- **Shell (Bash):** For writing utility scripts. The default shell is zsh, but bash is chosen for broader compatibility.
- **Git:** For version control.

## Development Setup
- A macOS environment.
- Visual Studio Code as the IDE.
- Standard command-line tools.

## Technical Constraints
- Scripts should be executable on macOS and potentially other Unix-like systems.
- Dependencies on external tools should be minimized for utility scripts.

## Dependencies
- Git must be installed and configured on the system.

## Tool Usage Patterns
- `git` command-line tool for version control operations.
- `chmod` for making scripts executable.
